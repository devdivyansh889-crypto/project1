# 📚 Library Management System

## 🔹 Description
This is a Python-based Library Management System that allows users to manage books efficiently.

## 🔹 Features
- Add Book
- Show Books
- Issue Book
- Return Book
- Fine Calculation System
- Day Tracking System

## 🔹 Technologies Used
- Python
- Dictionary (for storing book records)

## 🔹 How to Run
1. Open terminal
2. Run:
   python main.py

## 🔹 Project Structure
- main.py → Main menu
- add_books.py → Add books
- issue_book.py → Issue books
- return_book.py → Return books
- shows_book.py → Display books
- utils.py → Shared data
